JQuery的事件函数

一、jQuery的事件函数

1、基础事件

jQuer通过.bind()方法来为元素绑定事件，可以传递三个参数：bind(type,[data],function),

  • type:表示一个或多个类型的事件名字符串（click,dbclick,mousedown,mouseup,mousemove,mouseover,mouseout,change,select,submit,keydown,keypress,keyup,blur,focus,load,resize,scroll,error）
  • data:可选
  • 表示绑定到指定元素的处理函数

jQuery解除事件的绑定：unbind()

jQuery封装了很多的简写事件，可以直接调用不用再写bind了直接$(“input”).click(function(){})就可以了

方法名 说明 方法名 说明
click(fn) 鼠标单击 dbclick(fn) 鼠标双击
mousedown(fn) 鼠标点击后 mouseup(fn) 鼠标点击击弹起
mouseover(fn) 鼠标移入(会触发子节点) mouseout(fn) 鼠标移出(会触发子节点)
mousemove(fn) 鼠标移动 mouseenter(fn) 鼠标穿过(不会触发子节点)
mouseleave(fn) 鼠标穿出(不会触发子节点) keydown(fn) 键盘按下（返回建码)
keyup(fn) 键盘按下弹起 keypress(fn) 键盘按下()返回字符编码
unload(fn) 卸载本页面时要执行的函数 resize(fn) 文档改变大小
scroll(fn) 滚动条拖动 focus(fn) 表单焦点激活(子元素不会触发)
blur(fn) 表单焦点失去(子元素不会触发) focusin(fn) 表单焦点激活(子元素可触发)
focusout(fn) 表单焦点丢失(子元素可触发) select(fn) 表单文本选定
change(fn) 表单值改变 submit(fn) 表单提交(针对form标签)

2、复合事件

  • ready(fn)
  • hover([fn1,]fn2):鼠标移入触发fn1，移出触发fn2(mouseenter和mouseleave的结合体)
  • toggle(fn1,fn2[,fn3…]):已废弃，鼠标点击触发fn1,再次点击触发fn2

3、高级事件

3.1 trigger——模拟操作：模拟用户操作的行为

3.2 自定义事件：自己定义的事件，一般配合trigger使用

3.3 triggerHandler：(1)和trigger一样，都是模拟用户操作，但是她不会触发元素的默认行为。(2)triggerHandler只会匹配第一个元素，而trigger会匹配所有。(3)trigger返回jQuery对象，triggerHandler只会返回调用函数的返回值。(4)trigger在创建事件的时候会冒泡

3.4 命名空间

3.5 事件委托

  • live和die:（1.9已经被删除了，看看就好）

  • delegate()和undelegate

在dom中有很多元素绑定相同事件的时候，建议适用事件委托。

3.6 on,off和one

目前事件绑定和解绑的方式有三组六个，所以在1.7之后，推出on()和off()替代之前的绑定方法

  • 普通绑定：bind
  • 普通解绑：unbind
  • 事件委托：live，delegate
  • 解除委托：die，undelegate
  • 新绑定：on
  • 新解绑：off

二、jQuery的事件对象

这个参数e并不是我们收入传的参数，而是jQuery为我们传递的事件对象event;

1.evet对象的属性

属性名 描述
type 获取这个事件的类型，例如：click
tartget 获取触发元素的DOM，就是你点哪个就是哪个
data 获取事件调用时的额外数据
relatedTarget 获取离开进入的最相邻的DOM元素
currentTarget 获取监听元素的DOM，你绑定的哪个就是哪个，等同于this
pageX/pageY 获取相对于页面原点的坐标
screenX/screenY 获取显示屏幕位置的坐标
clientX/clientY 获取相对于页面视口的坐标
result 获取上一个相同事件的返回值
timeSatmp 获取事件触发的时间戳
which 获取鼠标左中右键，或键盘按键
altKey/shiftKey/ctrlKey/metaKey 获取是否按下了alt,shift,ctrl或meta键

2.事件对象的行为

冒泡行为：一个元素同时包含多个相同事件，就会产生冒泡行为。如何阻止冒泡行为？

  • e.stopPropagation();
  • e.isPropagationStopped();

默认行为：网页中元素在操作的时候会有自己的默认行为，比如右击文本框输入区域会弹出系统菜单等等。如何阻止默认行为？

  • e.preventDefault();
  • e.isDefaultPrevented();

如何阻止默认行为又阻止冒泡行为呢？

取消冒泡，并且取消该事件后续的事件处理（不包含默认行为）

  • stopImmediatePropagation()
  • isImmediatePropagationStopped()
